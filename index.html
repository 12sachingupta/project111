Nextmv platform overview

An introduction to building decision services with the Nextmv platform

Nextmv is a DecisionOps platform built for developers. It is a set of tools for building, testing, deploying, and operating decision models. Use Nextmv CLI for workflow management, Nextmv SDK for modeling and optimization, and Nextmv Cloud for managing production runs.
